    How to read this view
    • The primary amount card shows IO Reference Amount for Internal Order Perspective and Sales Order Amount for Sales Order Perspective.
    • RKB Actual Amount includes all Odoo RKB rows.
    • Product Item RKB is the normal product/material subset.
    • Non-Product / Service Item RKB includes valid budget, service, and adjustment rows.
    • ROP Excess Amount compares ROP Amount to RKB Actual Amount by value only for rows that have ROP; it is a review indicator, not automatically an error.
    • PO Excess Amount compares PO Amount to ROP Amount by value only for rows that have PO; it is a review indicator, not automatically an error.
    • Sales Order Perspective uses Sales Order Amount and SO RKB Kontribusi as operational estimates only, not gross profit, COGS, or accounting margin.
    • Non-product/service rows are valid operational data, not an error.
